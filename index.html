MAZE PROJECT

Maze project is a 3D game created by using raycasting. Raycasting is a rendering technique to create a 3D perspective in a 2D map.

ABOUT THE MAZE PROJECT

For the end of my ALX/Holberton School foundational trimesters, I had to make my own portfolio project. Learning about game creation is what motivated me to start working on Maze-Game. I was interested in game development and... wanted to acquire fresh approaches and strategies that might be used in software creation. In order to create this game, I choose to become familiar with the Raycasting tool and implement it using the SDL2 library. Being able to create a pseudo-3D graphic in early games like Wolfenstein 3D changed the gaming industry and paved the way for other gaming engines and technologies.

Keyboard Keys for Players

The keys below will help you to interact with the game

  • Arrow Keys -- Player Directions
  • M -- Map
  • R -- Refresh
  • C -- Zoom Player view
  • [ -- Zoom Out in Game
  • ] -- Zoom In in Game
  • F -- Change Views (Normal view to Panoramic view and vice versa)
  • T -- Change Texture of walls and square columns
